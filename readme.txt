# AI Health Prediction System

## Overview

This project is an AI-powered Health Prediction System developed using Python and Streamlit. The application allows healthcare staff to manage patient health records with CRUD operations and generate AI-based health remarks from blood test values.

## Features

* Add Patient Records
* View Patient Records
* Update Patient Information
* Delete Patient Records
* AI-based Health Prediction
* PDF Report Generation
* Email Validation
* Date Validation
* Persistent SQLite Database Storage
* Modern User Interface using Streamlit

## Technologies Used

* Python
* Streamlit
* SQLite
* Pandas
* ReportLab

## Installation

Install required libraries:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## AI Prediction Logic

The application predicts possible health risks based on:

* Glucose Levels
* Haemoglobin Levels
* Cholesterol Levels

## Database

SQLite database is used for persistent storage of patient records.
